MetaDL Challenge

During the challenge, each phase has its own data as a meta-dataset. Each meta-dataset is used to generate few-shot learning tasks via an API from the Meta-dataset repo. For more details, please refer to the provided tutorial.

Public data: The Omniglot dataset is provided during the Public phase. The dataset contains 1623 classes from 50 different alphabets. All images are of shape [28,28,3]. These characters were drawn online via Amazon's Mechanical Turk by 20 different people. The original dataset is presented in this page.

Feedback data: The data is kept hidden but you can get quick feedback on the leaderboard.

Final data: This phase will start at the very end of the challenge. The data is kept hidden.
